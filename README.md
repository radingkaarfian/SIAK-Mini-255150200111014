# Sistem Informasi Akademik Mini

This project is an academic assingment to demonstrate the application of the SCRUM framework using an example project. Every part of this project serves as sample code showcasing how to build the following:

## Project Overview

**Product Vision:** Create a simple system to manage relationships between Academic Advisors (Dosen) and their Supervised Students (Mahasiswa Bimbingan).

**Current Role:** Acting as the Development Team responsible for building and implementing the features of this Academic Information System.

## What This Project Demonstrates

* Create a new project in tree.taiga.io using SCRUM template for Agile project management
* Create many-to-many relationships between Dosen and Mahasiswa
* Implement proper class structures with attributes and methods
* Design Mahasiswa and Dosen classes based on real-world academic requirements


## Features

* **Dosen Management** - Create, read, update, and delete academic advisor records
* **Mahasiswa Management** - Manage student information and profiles
